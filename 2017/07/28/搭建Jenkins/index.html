Jenkins构建

安装Jenkins

在Mac下可以直接通过brew来安装Jenkins。

  • brew install jenkins

启动Jenkins

mac下使用brew services 来启动jenkins，使用launchctl设置开机启动Jenkins。

  • brew services jenkins start：启动
  • launchctl load -w /Library/LaunchDaemons/org.jenkins-ci.plist：设置开机启动

注意：Jenkins的开机启动配置文件/Library/LaunchDaemons/org.jenkins-ci.plist中，有一些值得注意的字段。

  • UserName字段：设置启动Jenkins的用户。
  • JENKINS_HOME字段：指定Jenkins的项目目录，所有Jenkins的配置和项目被存放在此处。
    一般情况下默认启动用户为Jenkins，其他用户没有权限写Jenkins的项目目录，这时就需要手动配置了。

关闭Jenkins

mac下使用brew services 来关闭jenkins，使用launchctl取消开机启动Jenkins。

  • brew services jenkins stop：关闭
  • launchctl unload -w /Library/LaunchDaemons/org.jenkins-ci.plist：取消开机启动

创建Jenkins Job

启动Jenkins后，可以浏览器中输入http://localhost:8080/直接进入Jenkins工程界面，首页中能直接新建Job。

拷贝Jenkins Job

将一台机器上Jenkins的Job拷贝到另外一条机器上Jenkins中，一般有三种办法。

  1. 拷贝Job目录。
  2. 利用Jenkins插件Jon Import Plugin
  3. 利用Jenkins命令工具Jenkins CLI去获得指定Job的XML文件，然后通过Jenkins CLI安装到另外一个Jenkins上。
